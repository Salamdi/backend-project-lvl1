## Setup

```sh
$ make install
```

## Usage

There are 5 games each of wich is run by the following command

```sh
$ make play GAME=*game-name*
```
where <*game-name*> is one of
- `brain-calc`
- `brain-even`
- `brain-gcd`
- `brain-prime`
- `brain-progression`

If you skip the `GAME` argument it just wlecomes you by your name.
